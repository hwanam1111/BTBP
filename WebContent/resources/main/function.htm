Features

중소기업에 특화된 다양하고 활용도 높은 기능들을 만나보세요!

관리자용 매뉴얼 사용자용 매뉴얼

  • TEAM
    팀

    BTBP을 시작하는 단위 "팀"

    BTBP 처음 시작할 때, 팀을 생성해야 합니다.
    하나의 팀은 한 개의 회사 혹은 한 개의 프로젝트를 의미합니다.
    팀 별로 하나의 조직도와 메신저, 그룹피드를 공유할 수 있습니다.

  • TEAM
    팀

    조직을 위한 팀 생성

    조직이 원하는 형태로 팀을 생성할 수 있도록 지원합니다.
    원하는 보안 수준에 따라, 관리자만이 멤버를 초대할 수 있는
    수직적 조직 혹은 멤버 누구나 초대할 수 있는 수평적 조직을
    생성할 수 있습니다. 또한 초대 가능한 도메인을 설정할 수 있어,
    사내 커뮤니케이션 플랫폼만으로도 사용할 수 있습니다.

  • TEAM
    팀

    외부 업체와의 협업도 TeamUP으로

    외부 협력사 등 외부인과의 업무 커뮤니케이션이 필요할 때,
    TeamUP을 사용해보세요. 관련 담당자를 초대하여
    제약 없는 소통과 즉각적인 협업이 가능해집니다.

  • GROUPFEED
    그룹피드

    그룹별로 소통하는 “그룹피드”

    그룹피드는 그룹피드의 멤버들이 자유롭게 사용하는 소통의 장입니다.
    목적에 따라, 전사게시판, 친목게시판, 프로젝트 게시판, 개인 게시판
    등으로 다양하게 활용할 수 있어 한곳에 업무를 집중할 수 있습니다.
    멤버간의 의견 교류와 프로젝트 자료 공유 등 피드글 작성을 통해
    그룹의 효율적인 협업을 지원합니다.

  • GROUPFEED
    그룹피드

    조직을 위한 팀 생성

    조직이 원하는 형태로 팀을 생성할 수 있도록 지원합니다.
    원하는 보안 수준에 따라, 관리자만이 멤버를 초대할 수 있는
    수직적 조직 혹은 멤버 누구나 초대할 수 있는 수평적 조직을
    생성할 수 있습니다. 또한 초대 가능한 도메인을 설정할 수 있어,
    사내 커뮤니케이션 플랫폼만으로도 사용할 수 있습니다.

  • GROUPFEED
    그룹피드

    협업을 위한 풍부한 피드글 기능

    파일 첨부, 캡처, 강제 알림 등을 활용하여, 피드글을 작성해보세요.
    피드글은 고유의 글번호가 있어, #글번호 태그로 공유할 수 있습니다.
    피드글이나 댓글로 @동료를 태그하면 동료에게 알림을 전달됩니다.
    서식 편집, 이모티콘까지 활용해 피드글을 작성 할 수 있습니다.

  • MESSENGER
    메신저

    소통이 필요할 때, 동료와 대화해보세요!

    메신저로 언제든지 동료와의 대화를 시작할 수 있습니다.
    1:1대화 뿐만 아니라 단체 대화까지 지원합니다.
    동료가 PC인지 모바일인지 접속상태도 확인하고
    상태메시지도 확인할 수 있어요

  • MESSENGER
    메신저

    편리한 메신저 기능 지원

    대화창에서 사진, 동영상, 파일, 캡처 기능을 지원합니다.
    P2P 대용량 파일 전송부터 모바일 미리보기까지
    PC와 모바일에서 모두 편리하게 사용하세요.
    파일 모아보기를 통해, 주고 받은 파일을 확인하고
    대화 검색으로 대화 내용을 찾아 볼 수 있습니다.

  • MESSENGER
    메신저

    신속한 대화를 위한 검색 기능

    동료를 검색하고, 대화를 시작할 수 있습니다.
    빠른 채팅 옵션(CapsLock키 두번)을 사용해
    빠르게 동료를 찾아 대화해보세요.

  • SHARED FILES
    모아보기

    그룹피드, 채팅에서 공유한 파일을 모아보기!

    그룹피드와 채팅에서 공유한 문서를 한번에 모아보세요!
    모아보기를 통해, 공유 히스토리를 파악할 수 있습니다.

  • SHARED FILES
    모아보기

    공유한 파일은 BTBP클라우드에 저장까지!

    모아보기에서 열람한 공유 파일은
    BTBP 클라우드에 저장할 수 있습니다.

  • MY CLOUD
    마이 클라우드

    안전하고 간편한 파일 공유!

    PC, 모바일에서 언제 어디서나
    BTBP 클라우드를 통해 내부 자료를 안전하고 간편하게 공유하고
    조직 내에서 효율적인 협업으로 팀워크를 더욱 강화할 수 있습니다!

  • MY CLOUD
    마이 클라우드

    필요한 만큼 용량 제공

    BTBP 클라우드는 협업에 필요한 용량을 지원합니다.
    BTBP 프리미엄으로 성장하는 조직에 필요한 넉넉한 용량을 이용하세요.

  • MY CLOUD
    마이 클라우드

    가시성 도구로 편리한 관리

    관리자에게는 가시성 도구를 제공하여
    회사 데이터를 감독할 수 있습니다.

  • ORGANIZATION
    조직도

    언제 어디서나 사내 주소록으로

    조직도 기반으로 제공되어 구성원 정보를 쉽게 검색하고
    즉시 연락할 수 있습니다.
    모바일에서도 담당자의 근무처와 업무까지 확인할 수 있습니다.
    필요 시 신속히 대화하고, 직통번호를 통해 바로 연락하세요.

  • ORGANIZATION
    조직도

    동료의 상태를 표시합니다

    조직도를 활용해서, 동료의 상태를 확인해보세요!
    동료가 PC로 접속중인지 모바일로 접속중인지 접속 상태를 쉽게
    알 수 있습니다.

  • ORGANIZATION
    조직도

    동료의 정보를 편리하게 열람

    프로필 정보를 통해 동료의 프로필과 상태명을 확인하세요.
    직급, 직책, 소속, 직통번호, 휴대폰 번호, 이메일, 생일정보 등
    프로필 정보부터 동료가 작성한 글 모아보기까지 동료에 관련된
    상세한 정보를 확인 할 수 있습니다.

  • FILE
    파일

    자유로운 파일 전송

    파일 확장자에 관계없이, PC와 모바일간 자유로운 파일전송이
    가능합니다.
    같은 팀/ 동료끼리 어느 디바이스에서나 파일을 교류할 수 있습니다.
    그룹피드와 메신저를 통해 외부미팅, 출장 등 이동중에도
    자유롭게 자료를 확인하고 공유 할 수 있습니다.

  • FILE
    파일

    편리한 파일 첨부와 미리보기

    Drag&Drop으로 Window탐색기에서 간편하게 파일을 첨부하세요!
    게시글 작성시 또는 메시지 창에서 화면을 캡처하고 바로 전송도
    가능합니다. 모바일에서 문서파일 미리보기, 다운로드 파일목록 보기
    기능으로 편리하게 다운받은 파일을 확인할 수 있도록 도와줍니다.

  • FILE
    파일

    대용량 파일 전송(P2P)

    메신저에서 대용량파일전송 기능을 지원합니다.
    한번에 10개 이상의 파일을 전송, 폴더 전송, 100MB 이상의 파일을
    전송할 때는 대용량파일전송 모드로 파일을 전송합니다

  • SEARCH
    검색

    통합검색으로 쉽고 빠르게 찾아보세요!

    게시글뿐만 아니라 대화내용까지 검색이 가능하여
    수많은 자료들 중 원하는 자료를 쉽고 빠르게
    찾아볼 수 있습니다.

  • SEARCH
    검색

    편리한 검색 결과 제공

    검색 결과 리스트와 상세화면을 함께 볼 수 있어
    원하는 결과를 편하게 찾아 볼 수 있습니다.

  • NOTIFICATION
    알림

    어떤 소식에도 빠르게 반응할 수 있어요

    알림팝업과 알림창으로 소식에 빠르게 반응할 수 있습니다.
    PC뿐만 아니라 모바일로도 알림을 받아 볼 수 있어 중요한 소식을
    놓치지 않도록 도와줍니다.

  • NOTIFICATION
    알림

    자리비움모드로, 알림 호환 지원

    PC버전을 사용하고 있을때, 일정 시간 PC를 사용하지 않으면
    “자리 비움 모드”로 변경되며, 이후 알림은 모바일로 알림을 보내줍니다.
    설정 탭에서 자리 비움 시간을 설정하고, 알림을 받아보세요!

  • ADMIN
    관리자

    슈퍼관리자 기능으로 팀을 일괄적으로 관리

    TeamUP은 팀마다 독립적인 서비스로 제공되며,
    각 “팀”의 슈퍼관리자는 서비스 사용을 일괄적으로 관리할 수 있습니다.
    팀프로필, 관리자, 동료, 로그, 팀초대 등 다양한 관리자 기능으로
    편리한 팀관리를 경험해보세요.

  • ADMIN
    관리자

    조직도 기반의 편리한 팀 운영

    조직도 기반으로 사용할 수 있는 기업용 커뮤니케이션 툴입니다.
    조직도 기반으로 사용자를 관리하고,
    여러 명의 관리자를 두어 편리하게 “팀”을 운영해보세요.

  • OPEN API
    오픈 API

    사내서비스에 TeamUP을 연동할 수 있습니다.

    회사 내 서비스의 API가 있다면 BTBP과 연동하여 사용이 가능합니다.
    OpenID 표준인증방식 적용으로 보안 걱정 없이,
    사내 서비스를 TeamUP과 연동하여 편리하게 사용해보세요!

  • OPEN API
    오픈 API

    외부 APP 연동 및 알림(준비중)

    Trello, 각종 드라이브와 같은 외부 협업관리툴을
    연동하고 알림을 받을 수 있습니다.
